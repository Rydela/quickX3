Overview

HXLDash is a dashboard and online mapping tool designed for humanitarians and humanitarian contexts. HXLDash's aim is to make creating dashboards possible in less than 2 minutes by leveraging the power of the Humanitarian Exchange Language and linking to the common operation datasets.

HXLDash does not aim to replace popular, powerful and fully featured dashboarding tools such as Tableau or PowerBI, but rather be a complimentary tool with features built with humanitarians in mind. You can read more about the unique features that make it suitable for humanitarians here

It is currently in an alpha stage and we are looking to test the tool with different users to work out what is useful and what is not as we plan to add new features.

HXLDash is used in production by the IFRC. Look at the examples section for more cases studies.
